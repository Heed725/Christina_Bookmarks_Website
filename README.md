# Christina's Bookmarks Website

A curated, searchable collection of bookmarks organized by topic — covering GIS, Remote Sensing, Conservation, Academic Research, and more.

**Live site:** [https://Heed725.github.io/Christina_Bookmarks_Website/](https://Heed725.github.io/Christina_Bookmarks_Website/)

## Categories

- **GIS & Remote Sensing** — Tools, tutorials, and resources for GIS professionals
- **Google Earth Engine** — Scripts, datasets, and apps for cloud-based geospatial analysis
- **Remote Sensing** — Satellite missions, image processing, and analysis techniques
- **Geospatial Data Sources** — Free datasets for climate, population, biodiversity, and more
- **Conservation & Wildlife** — IUCN, camera trap AI, and conservation technology
- **Human-Wildlife Conflict** — Research papers and strategies for HEC management
- **Academic Resources** — Journals, databases, and research tools
- **Career & Jobs** — GIS job boards and professional profiles
- And more!

## Built With

- [MkDocs](https://www.mkdocs.org/) with [Material theme](https://squidfunk.github.io/mkdocs-material/)
- Organized with the help of [Claude Code](https://claude.ai/code)
- Deployed via GitHub Pages

## How to Update

1. Edit `parse_bookmarks.py` to add new bookmarks
2. Run `python3 parse_bookmarks.py` to regenerate the site content
3. Commit and push — GitHub Actions will auto-deploy
